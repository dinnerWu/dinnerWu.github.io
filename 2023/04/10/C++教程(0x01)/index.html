C++教程(0x01)


C++是什么？

C++ 是一种高级语言，它是由 Bjarne Stroustrup 于 1979 年在贝尔实验室开始设计开发的。C++ 进一步扩充和完善了 C 语言，是一种面向对象的程序设计语言。C++ 可运行于多种平台上，如 Windows、MAC 操作系统以及 UNIX 的各种版本。

（by 菜鸟编程）

#include<bits/stdc++.h>：每个源文件里必须加入的，include(引入)，bits/stdc++.h(万能头文件)

变量

变量是一种可变的量，就像一个盒子，可以装东西，变量这个盒子可以装很多东西，数字、字符……

数据类型

操作系统会根据变量的数据类型，来分配内存和决定在保留内存中存储什么。

整数型

整形类型的标识符为int。根据取值范围分为以下几种：

数据类型 定义标识符 占字节数 数值范围(二的次方表示)
短整型 short [int] 2(16位) -2^15~2^15-1
整型 [long] int 4(32位) -2^31~2^31-1
长整型 long [int] 4(32位) -2^31~2^31-1
超长整型 long long [int] 8(64位) -2^63~2^63-1
无符号整型 unsigned [int] 2(16位) 0~2^16-1
无符号短整型 unsigned short [int] 2(16位) 0~2^16-1
无符号长整型 unsigned long [int] 4(32位) 0~2^32-1
无符号超长整型 unsigned long long 8(64位) 0~2^64-1

实数（小数）型

一个实数形数据用来存储实数，包括正实数、负实数和0

表示法

十进制表示法：

例：0.1，-0.01，9.23

科学计数法：

例：1.25E+05(1.25*10^5)


实数型有以下几种数据类型：

数据类型 定义标识符 数值范围
单精度实型 float -3.4E-38~3.4E+38
双精度实型（常用） double -1.7E+308~1.7E+308
长双精度实型 long double -3.4E+4932~1.1E+4932

字符型

字符型只有一种，就是char，char用ascii存储字符。

定义

变量的定义是数据类型 变量名1,变量名2...;

加+、减-、乘*、除/（当除法的两个数为整型时，会直接舍去小数部分）、取余%（取模，指求除法的余数）、自增++（变量加一）、自减–（变量减一）共7种

2.

大于>、小于<、等于==、大于等于>=、小于等于<=、不等于!= 共6种

3.

与&&、或||、非! 共3种

4.（初学者不用学）

按位与&、按位或|、按位非~、按位异或^、左移<<、右移>> 共6种

简单赋值=、复合算术赋值+=,-=,*=,/=,%=（“a·=x"与"a=a·x"是一样的意思，”·"代表复合算数赋值等于号前面的符号，"x"是值）、复合位运算赋值&=,|=,^=,>>=,<<=（“a·=x"与"a=a·x"是一样的意思，”·"代表复合位运算赋值等于号前面的符号，"x"是值）共3种

6.

三目运算符?:（用于条件求值，例如"a>=10? a-10:a"的意思就是如果a>=10，答案就是a-10，否则就是a）共1种

7.

8.（初学者不用学）

取内容*，取地址& 共2种

9.

求数据类型所占的字节数sizeof 共1种

练习题

1.include是什么意思( )

A.在俱乐部里 B.引入

2.a=10,a-=5运行这段程序后，a的值是多少( )

A.10 B.5

*3.变量a是整型(int)，sizeof(a)的结果是多少( )

A.4 B.2

答案（做完了才能看）

  1. B（include的意思是引入）
  2. B（a=10是将a赋值为10，a-=5的意思是将a减去5，所以a的值就是10-5=5）
  3. A（变量a是整型，而整形占4个字节，所以结果是4）
